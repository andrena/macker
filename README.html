About This Distribution (@dist.version.long@)

In this distribution, you'll find:

build/macker.jarMacker binary, ready for rule-checking fun
lib/libraries required by Macker
src/Macker source code
doc/lots of fascinating documentation, including an FAQ and a user guide
bin/useful scripts for running Macker from a command line
LICENSE.htmllicense for Macker (GPL)
lib/license/licenses for bundled libraries

Not quite sure what Macker is? Read the FAQ.

Already using Macker? Here's an overview of what's new in version 0.4.

This distribution includes a local copy of the Macker site. Check the online version for updates.
